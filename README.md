# 🎲 Dice Game
A fun two-player dice game built with vanilla HTML, CSS, and JavaScript. Each player rolls a dice and the one with the highest number wins the round!or it might even end up as a draw!!


🔗 [Play the Game!](https://13siri06.github.io/DiceGame/)
---
## 🎮 How to Play

1. Open the game in your browser
2. Click the **Roll Dice 🎲** button
3. Both players roll simultaneously
4. The player with the **higher number wins** 🏆
5. If both roll the same number — it's a **Draw!** ⚖️

---

## 🛠️ Built With

- **HTML** — Structure and layout
- **CSS** — Styling and design
- **JavaScript** — Game logic and dice randomization and
I  also learned DOM manipulation with JavaScript

---

## ⚙️ How It Works

- User clicks the **Roll Dice** button to start each round
- `Math.random()` generates a random number between 1–6 for each player
- The corresponding dice image updates dynamically in the DOM
- The result is compared and the winner is displayed instantly on screen
